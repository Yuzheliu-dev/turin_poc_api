# RAVEN API Documentation

API documentation for **RAVEN** (Reasoning Agent for Vulnerability ExaminatioN) — an AI-driven platform for automated code vulnerability detection and remediation.

## Build locally

```bash
pip install -r requirements.txt
make html
# open build/html/index.html
```

## Deploy

This project is configured for [Read the Docs](https://readthedocs.org/). Push to `main` to trigger a rebuild.
