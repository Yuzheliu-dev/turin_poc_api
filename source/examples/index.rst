Code Examples
=============

This section provides ready-to-use code examples for every RAVEN API endpoint in three
languages:

- :doc:`curl` -- Command-line examples using cURL
- :doc:`python` -- Python examples using the ``requests`` library
- :doc:`javascript` -- JavaScript examples using the ``fetch`` API

All examples assume your API key is stored in the ``RAVEN_API_KEY`` environment variable:

.. code-block:: bash

   export RAVEN_API_KEY="your-api-key-here"
